Any VPS Hosting (Generic Steps)
1) Ubuntu 22.04 LTS server → update: sudo apt update && sudo apt -y upgrade
2) Install Node.js 18+ & Git → clone your SHABAN-MD repo
3) Set env vars / session ID securely (use .env)
4) Install deps: npm ci or npm install
5) Start with a process manager: pm2 start npm --name shaban-md -- start
6) (Optional) Reverse proxy with Nginx + SSL (Certbot) for web dashboards/webhooks
